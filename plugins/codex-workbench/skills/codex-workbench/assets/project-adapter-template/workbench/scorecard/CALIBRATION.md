本文件用于校准 `scorecard.py` 的评分口径。分数只代表证据成熟度和流程一致性，不能证明业务正确、架构合理、安全合规或 AI 输出可信。

## 使用规则

- 没有锚定样例和人工抽查时，分数可信度只能视为低或中。
- `full` 档必须完成校准，不能只靠总分通过。
- 发现误报、漏报或被刷分时，优先修改模板、测试、质量门、CI、hook 或审查清单。
- 不允许为了提高分数而补空文档、假状态或无验证证据。
