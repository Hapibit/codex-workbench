## 可访问性和可用性

- [ ] 文案能被目标用户理解。
- [ ] 关键按钮、表单、错误提示不依赖开发者解释。
- [ ] 移动端和桌面端不遮挡、不裁切、不横向溢出。
- [ ] 高风险操作有确认、撤销或明确后果提示。
